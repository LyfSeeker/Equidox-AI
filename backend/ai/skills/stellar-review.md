# Stellar Network Review

## Best practices
- Correct network passphrase
- SAC for native asset
- User keys in Freighter
- Testnet vs Pubnet clarity

## Common mistakes
- Hardcoded mainnet in demos
- Ignoring fees/sequence

## Red flags
- Weak custodial key storage

## Evaluation checklist
- [ ] Horizon/RPC URLs
- [ ] Contract IDs
- [ ] Friendbot only on testnet

## Scoring guidance
Ops correctness for chain integration.

## Example observation
Docs say Testnet; config differs — mismatch evidence.
