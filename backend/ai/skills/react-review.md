# React Review

## Best practices
- Clear component ownership
- Hooks rules honored
- Controlled forms
- Loading and error states

## Common mistakes
- useEffect for derived state
- Mutating props/state
- God components

## Red flags
- Secrets in client bundle
- Unsafe HTML injection

## Evaluation checklist
- [ ] Client vs server boundary
- [ ] List keys
- [ ] Auth-gated screens

## Scoring guidance
Score structure and data-flow clarity.

## Example observation
Forms write unvalidated data to storage — note risk.
