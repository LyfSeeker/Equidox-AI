# Grant / Milestone Review

## Best practices
- Measurable criteria
- Evidence maps 1:1 to criteria
- Payout proportional to delivery

## Common mistakes
- Vague criteria scored generously
- Paying on promises

## Red flags
- Ignoring criteria for a nice README

## Evaluation checklist
- [ ] Each criterion status
- [ ] Amount vs complexity

## Scoring guidance
Feature Completion is primary; others support trust.

## Example observation
Criteria require Soroban deploy; only frontend present — FAIL.
